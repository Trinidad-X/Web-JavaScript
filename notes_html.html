04 Tables

Used for structured data (not layout!).

Key Attributes: colspan (merge columns), rowspan (merge rows).

Tag Description
<tr>Table Row
<th>Table Header (Bold/Centered)
<td>Table Data (Standard cell)
05 Forms & Inputs

The Mechanism: Data is sent from Client to Server.

GET vs POST:

  • GET: Data in URL (Insecure, book-markable). Good for search.
  • POST: Data in Body (Secure, hidden). Good for passwords.
06 Multimedia

Native support for Audio and Video without Flash.

<video controls poster="thumb.jpg">
  <source src="movie.mp4" type="video/mp4">
</video>

Attributes: controls, autoplay, loop, muted.
