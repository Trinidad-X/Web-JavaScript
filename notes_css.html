02. The Box Model (Visualized)

In CSS, every element is a box. The browser calculates size based on these layers:

Margin (Space outside)
Content
  • Content: The text or image.
  • Padding: Space between content and border (Green).
  • Border: The line around the padding (Yellow).
  • Margin: Space outside the border (Orange).
03. Selectors & Specificity

When multiple rules apply to the same element, the one with higher "Specificity Score" wins.

SelectorScoreType
style="..."1000Inline Style
#id0100ID

05. Flexbox & Grid

Modern Layout systems (replacing old "float" hacks).

  • Flexbox: One-dimensional (Row OR Column). Good for navbars, centering.
  • Grid: Two-dimensional (Rows AND Columns). Good for page skeletons.
06. Colors & Units
  • Hex: #ff0000 (Red)
  • RGB: rgb(255, 0, 0)
  • RGBA: rgba(255, 0, 0, 0.5) (50% opacity)

Relative Units (Recommended):

  • rem: Relative to root HTML size (usually 16px).
  • em: Relative to parent font size.
  • vh/vw: Viewport Height/Width %.
